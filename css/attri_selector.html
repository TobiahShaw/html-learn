具有特定属性的HTML元素样式

具有特定属性的HTML元素样式不仅仅是class和id。

下面的例子是把包含标题（title）的所有元素变为蓝色：

[title] { color:blue; }

下面的实例改变了标题title='tobiah'元素的边框样式:

[title=tobiah] { border:5px solid green; }

下面是包含指定值的title属性的元素样式的例子，使用（~）分隔属性和值:

[title=~runoob] { color:blue; }

下面是包含指定值的lang属性的元素样式的例子，使用（|）分隔属性和值:

[lang|=en] { color:blue; }
